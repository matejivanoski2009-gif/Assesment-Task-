# One Page Report:
## Purpose, Design Decisions, and Functionality:

## Purpose:
The purpose of this website is to improve student understanding of algorithms by visually illustrating how they run. Lots of students struggle to grasp loops and comparisons by reading code. This interactive system allows them to see each step take place in real time.

## Design Decisions:
The following coding languages will be utilised:
- HTML
- CSS
- JavaScript
- Python
