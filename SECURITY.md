# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x  | :white_check_mark: |
| < 1.0   | :x:                |

This project is intended for learning and demonstration purposes.

## Reporting a Vulnerability

If you discover a security issue in this Algorithm Learning Quiz App, please report it by creating a GitHub Issue in this repository.

When reporting a vulnerability, please include:
- Steps to reproduce the problem
- A concise description of the issue
- If possible, any screenshots 

Once a report is submitted, it will be reviewed and if the issue is confirmed, fixes will be commited.
